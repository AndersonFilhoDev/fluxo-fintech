# Fluxo — Dashboard Principal

Recriação da tela **Dashboard Principal (Visão Geral)** do app fictício **Fluxo**, uma fintech voltada para estudantes universitários.

Atividade de Web Development — FIAP.

## Sobre o projeto

A tela apresenta:

- **Saldo disponível do mês**, com barra de progresso da meta de economia.
- **Despesas por categoria**, em um gráfico de rosca (donut) feito 100% em CSS.
- **Orçamentos do mês**, com ícones e barras de progresso por categoria.
- **Últimos lançamentos**, com identificação de receitas e despesas.
- Botão flutuante para **adicionar lançamento**.

O layout é totalmente responsivo: em telas pequenas a barra lateral vira um cabeçalho superior e os cartões se reorganizam em uma única coluna.

## Tecnologias

- HTML5 semântico
- CSS3 (arquivo separado, `styles.css`)
- Tailwind CSS (via CDN)

Sem JavaScript e sem etapa de build.

## Como executar

Basta abrir o arquivo `index.html` diretamente no navegador (duplo clique). Não é necessário instalar nada nem rodar nenhum servidor.
